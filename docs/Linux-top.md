## 分项说明

### 开机信息

```txt
._________________________________________________________________________________.
| top - ##:##:## up ### days, ##:##, ## users,  load average: #.##, #.##, #.##    |

  |_/   |______/ |________________/  |______/   |____________________________/
  |     |        |                   |          过去 1 分钟、5 分钟、15 分钟平均负载
  |     |        |                   当前用户数
  |     |        系统已运行时间
  |     当前时间
  table od processes
```

         总进程数

           |        |        |        |        |        |        |        管理程序窃取的运行时间占比（stolen）
