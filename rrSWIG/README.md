## 指南

*   手动安装，并使用 CMake 进行链接（Windows）
    1.  访问 SWIG 官网，下载 `swigwin-4.0.0.zip` 包并解压
    2.  设置环境变量 `SWIG_HOME`
    3.  通过 CMake 的 `FindSWIG` 模块链接使用 SWIG 库
